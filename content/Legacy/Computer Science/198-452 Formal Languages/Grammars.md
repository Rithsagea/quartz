## Context Free Grammar

A <u>context-free grammar</u> is a 4-tuple $G=(V,\Sigma,R,S)$
- $V$ a finite set of variables
- $\Sigma$ a finite set disjoint from $V$ of terminals
- $R$ a set of rules, consisting of a variable and a string of variables and terminals
- $S\in V$ is the start variable

Given strings $u,v,w$ and a <u>rule</u> $A\to w$, $uAv$ yields $uwv$, or $uAv\Rightarrow uwv$
$u$ <u>derives</u> $v$ or $u\overset{*}\Rightarrow v$ if $u=v$ or there are $u_1,u_2,\ldots,u_k$ $k\ge0$ where
$$
u\Rightarrow u_1\Rightarrow u_2\Rightarrow\ldots\Rightarrow u_k\Rightarrow v
$$

The <u>language</u> of the grammar is $L(G)=\{w\in\Sigma^*,S\overset*\Rightarrow w\}$. A <u>context-free language</u> is a language that can be generated by a CFG.

Any regular language can be turned into a context free language. For arrow from state $a\to b$ with character $w$, add rule $R_a\to wR_b$

A derivation is a <u>left-most derivation</u> if at every step, the leftmost terminal is expanded first. A grammar is <u>ambiguous</u> if multiple left-most derivations can generate the same string

## Chomsky Normal Form

A CFG is in <u>Chomsky normal form</u> if every rule is of the following form, where $a$ is a terminal and $A,B,C$ are variables, and $B,C$ are not the start variable. Furthermore, $S\to\varepsilon$ is allowed
$$
A\to BC\qquad A\to a
$$

Every CFG can be written in Chomsky normal form
1. Create a new start variable $S_0\to S$
2. For all $A\to\varepsilon$, remove this rule. Then for all $R\to uAv$, add $R\to uv$ (note that $u,v$ can contain $A$). Iterate
3. For all $A\to B$, remove this rule. Whenever $B\to u$ appears, add $A\to u$ unless this was a previously removed unit rule.
4. For all $A\to u_1u_2\cdots u_k$ with $k\ge 3$, add rules $A\to u_1A_1$, $A_i\to u_{i+1}A_{i+1}$ and $A_{k-2}\to u_{k-1}u_k$. Replace terminals $u_i$ with $U_i$ and add $U_i\to u_i$

## Pushdown Automata
A <u>pushdown automata</u> is a 6-tuple $(Q,\Sigma,\Gamma,\delta,q_0,F)$
- $Q$, a finite set of states
- $\Sigma$, a finite input alphabet
- $\Gamma$, a finite stack alphabet
- $\delta:Q\times\Sigma_\varepsilon\times\Gamma_\varepsilon\to\mathcal P(Q\times\Gamma_\varepsilon)$ a transition function
- $q_0\in Q$ the start state
- $F\subset Q$ the set of accept states

A pushdown automata accepts input $w=w_1w_2\cdots w_m$ where $w_i\in\Sigma_\varepsilon$ and there is a sequence of states $r_0,r_1,\ldots,r_m\in Q$ and strings $s_0,s_1,\ldots,s_m\in\Gamma^*$ such that $r_0=q_0$, $s_0=\varepsilon$, for $0\le i\le m-1$, $(r_{i+1},b)\in\delta(r_i,w_{i+1},a)$, where $s_i=at$ and $s_{i+1}=bt$ for $a,b\in\Gamma_\varepsilon$ and $t\in\Gamma^*$, and $r_m\in F$.

### CFLs are PDAs
Given a CFG $G=(V,\Sigma,R,S)$, define a PDA $P=(Q,\Sigma,\Gamma,\delta,q_{start},F)$. Given states $q$ and $r$ and symbols $a\in\Sigma_\varepsilon$, $s\in\Gamma_\varepsilon$. In order to transition from $q$ to $r$ by reading $a$, popping $s$, and reading the string $u=u_1\cdots u_l$, generate additional states and transitions such that
$$
\begin{array}{c}
\delta(q,a,s)\ni(q_1,u_l),\\
\delta(q_1,\varepsilon,\varepsilon)=\{(q_2,u_{l-1})\},\\
\delta(q_2,\varepsilon,\varepsilon)=\{(q_3,u_{l-2})\},\\
\vdots\\
\delta(q_{l-1},\varepsilon,\varepsilon)=\{(r,u_1)\}\\
\end{array}
$$

By this construction, writing $(r,u)\in\delta(q,a,s)$ means if $q$ is the state and $a$ is the next input symbol, then the PDA can read $a$, pop $s$ then push $u$ onto the stack and go on to state $r$.

Take $Q=\{q_{start},q_{loop},q_{accept}\}\cup E$, where $E$ is the set of additional states from the above construction. Let $\delta(q_{start},\varepsilon,\varepsilon)=\{(q_{loop},S,\text\textdollar)\}$.
1. If the stack has a variable, $\delta(q_{loop},\varepsilon,A)=\{(q_{loop},w)|(A\to w)\in R\}$
2. If the stack has a terminal, $\delta(q_{loop},a,a)=\{(q_{loop},\varepsilon\}$
3. If the stack is the empty $\text\textdollar$, $\delta(q_{loop},\varepsilon,\text\textdollar)=\{(q_{accept},\varepsilon)\}$

### PDAs are CFLs
Let PDA $P=(Q,\Sigma,\Gamma,\delta,q_0,\{q_{accept}\})$. Construct $G$ with $\{A_{pq}|p,q\in Q\}$ and start variable $A_{q_0,q_{accept}}$
1. For $p,q,r,s\in Q$, $u\in\Gamma$, and $a,b\in\Sigma_\varepsilon$, if $\delta(p,a,\varepsilon)$ contains $(r,u)$ and $\delta(s,b,u)$ contains $(q,\varepsilon)$, add the rule $A_{pq}\to aA_{rs}b$
2. For $p,q,r\in Q$, add $A_{pq}\to A_{pr}A_{rq}$
3. For $p\in Q$, add the rule $A_{pp}\to\varepsilon$


## Pumping Lemma for CFL
If $A$ is a CFL, then there is $p$ (pumping length) where if $s$ is a string in $A$ where $|s|\ge p$, then $s$ can be divided into 5 pieces, $s=uvxyz$ where
1. for each $i\ge 0$, $wv^ixy^iz\in A$
2. $|vy|>0$, and
3. $|vxy|\le p$

Given a derivation $A\overset*\Rightarrow uRz\overset*\Rightarrow uvRyz\overset*\Rightarrow uvxyz$, we see that $R\overset*\Rightarrow vRy$, and we can repeat $v$ and $y$ as many or as few times as needed. Note that this only happens if the length of the string is such that the height of the parse tree must exceed the number of variables.
