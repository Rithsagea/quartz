We are interested in approximating distance between $n$ points $P\subseteq\mathbb R^d$. For some sets $A,B$, we write $A\otimes B=\{\{x,y\}|x\in A,y\in B,x\ne y\}$ as all unordered pairs of points. Additionally, define $d(A,B)=\min\{\|a-b\|:a\in A,b\in B\}$

A <u>pair decomposition</u> of $P$ is a set of pairs $\mathcal W=\{\{A_1,B_1\},\ldots,\{A_m,B_m\}\}$ such that
1. For every $i\in[m]$, $A_i,B_i\subset P$
2. For every $i\in[m]$, $A_i\cap B_i=\varnothing$
3. $\bigcup_{i=1}^sA_i\otimes B_i=P\otimes P$

A pair $A,B$ is $1/\varepsilon$ <u>well-separated</u> if $\max(\diam A,\diam B)\le\varepsilon\cdot d(A,B)$. A $1/\varepsilon$ <u>well-separated pair decomposition</u> (WSPD) is a set of pairs $\mathcal W=\{\{A_1,B_i\},\ldots,\{A_m,B_m\}\}$ where each $A_i$ and $B_i$ is $1/\varepsilon$ well-separated.

# Quadtree Construction
Given a point set $P\subseteq\mathbb R^d$, construct a [[Quadtree#Path Compression|compressed quadtree]] $\mathcal T$ with root $u_0\in\mathcal T$. For some node $u\in\mathcal T$, write $\square_u$ as the corresponding hypercube. Define $\Delta(v)=\diam\square_v$ and $P_v$ as the set of points in $v$. 
For two nodes $u,v$, we write $d(u,v)=d(\square_u,\square_v)$.

To compute the $1/\varepsilon$-WSPD of $P$, call $\mathsf{algWSPD}(u_0,u_0)$.

>[!abstract] $\textsf{algWSPD}(u,v,s)$
>1. If $u=v$ and $\Delta(u)=0$, return
>2. If $\Delta(u)<\Delta(v)$, swap $u$ and $v$
>3. If $\Delta(u)\le \varepsilon\cdot d(u,v)$, return $\{\{u,v\}\}$
>4. Return $\bigcup^r_{i=1}\mathsf{algWSPD}(u_i,v)$ where $u_i$ are children of $u$

## Lemma 1
>[!info]
>For some pair $\{u,v\}\in\mathcal W$ computed by $\mathsf{algWSPD}$, we have $\max(\Delta(u),\Delta(v))\le\min(\Delta(\bar p(u)),\Delta(\bar p(v)))$, where $\bar p(x)$ is the parent node of $x$ in $\mathcal T$. ^tt1wce

A moment's thought shows that $\Delta(u)\le\Delta(\bar p(u))$ and $\Delta(v)\le(\bar p(v))$. At some point, we call $\mathsf{algWSPD}(u,v)$ through a sequence of arguments $\{(u_0,u_0),(u_1,v_1),\ldots ,(u_s,v_s):=(u,v)\}$. WLOG assume that $u_{s-1}=u_s$ and $v_{s-1}=\bar p(v_s)$. Take $t\in[s]$ to be the last index where $u_{t-1}=\bar p(u)$. Note $v$ is a descendent of $v_{t-1}$ and the larger cell is always refined first, thus $\Delta(v)\le\Delta(v_t)=\Delta(v_{t-1})\le\Delta(u_{t-1})=\Delta(\bar p(u))$. By symmetry this argument holds for $\Delta(u)\le\Delta(\bar p(v))$.

## Lemma 2
>[!info]
>Let $\square$ be a cell of a grid $G$ of $\mathbb R^d$ with diameter $x$. For $y\ge x$, there are at $O((y/x)^d)$ cells of distance at most $y$ from $\square$ ^dzeiuj

Every cell with distance at most $y$ from $\square$ is contained in some hypercube with side length $2y$. This hypercube has volume $(2y)^d$, while $\square$ has volume $(x/\sqrt d)^d$. Thus by packing we can fit at most $O((y/x)^d)$ cells in this hypercube, bounding the number of candidate cells.

## Lemma 3
>[!info]
>For the WSPD generated by $\mathsf{algWSPD}$, for any pair $\{u,v\}$ and $x\in P_u$, $y\in P_v$, we have $d(u,v)\le\|x-y\|$

Note that $d(u,v)=d(\square_u,\square_v)\le d(P_u,P_v)\le \|x-y\|$, since $P_u\subseteq\square_u$ and $P_v\subseteq\square_v$.

## Correctness

Performing structural induction on $\mathsf{algWSPD}(u,v)$, it is clear that this will produce a pair of subsets which covers every pair in $P_u\otimes P_v$. Thus $\mathsf{algWSPD}(u_0,u_0)$ covers every pair of $P$.

For some output pair $\{u,v\}$, if both are singletons it is clear that $P_u$ and $P_v$ are well separated. Otherwise $\Delta(u)=\max(\Delta(u),\Delta(v))>0$, so $d(P_u,P_v)\ge d(u,v)\ge\Delta(u)/\varepsilon>0$. Therefore $P_u\cap P_v=\varnothing$ and this is a valid pair decomposition.

For every output pair $\{u,v\}$, by construction we have $\max(\diam P_u,\diam P_v)\le\max(\Delta(u),\Delta(v))\le\varepsilon\cdot d(u,v)$. Thus we have a $1/\varepsilon$-WSPD.

## Complexity

The quadtree can be constructed in $O(n\log n)$. The remainder of the algorithm is linear in output size, thus it suffices to bound the number of pairs returned.

For some pair $\{u,v\}$ generated by $\mathsf{algWSPD}(u,v)$, suppose this was called by $\mathsf{algWSPD}(u,v')$, where $v'=\bar p(v)$ is the parent of $v$. Charge the cost of computing $\{u,v\}$ to $v'$. For all such $v'$ in $\mathcal T$, any pair $\{u,v'\}$ was considered but not added, thus $\Delta(v')>\varepsilon\cdot d(u,v')$, so $d(u,v')<r:=\Delta(v')/\varepsilon$. By [[Well Separated Pair Decomposition#Lemma 1|Lemma 1]], $\Delta(\bar p(u))\ge\Delta(v')\ge\Delta(u)$, thus we case on equality.

<u>Case 1</u> $\Delta(v')=\Delta(u)$. By [[Well Separated Pair Decomposition#Lemma 2|Lemma 2]], there are at most $O((r/\Delta(v'))^d)=O(1/\varepsilon^d)$ candidates for $u$. Since $v'$ has at most $2^d$ children, this charge occurs at most $O(2^d\cdot1/\varepsilon^d)$ times.

<u>Case 2</u> $\Delta(\bar p(u))=\Delta(v')$. Note $d(\bar p(u),v')\le d(u,v')<r$, thus by identical argument, there are $O(1/\varepsilon^d)$ candidates for $\bar p(u)$. This node also has at most $2^d$ children, so we have at most $O(2^d\cdot 2^d\cdot1/\varepsilon^d)$ charges.

<u>Case 3</u> $\Delta(\bar p(u))>\Delta(v')>\Delta(u)$. Consider the grid of cells about $\square_{v'}$, and let $\hat\square$ be the cell containing $\square_u$. Note that $\square_u\subsetneq\hat\square\subsetneq\square_{\bar p(u)}$, so $d(\hat\square,\square_{v'})\le d(\square_u,\square_v)=d(u,v')<r$. Similarly, there are $O(1/\varepsilon^d)$ candidates of $\hat\square$. Since we only consider immediate children of $\hat\square$, there are at most $O(2^d\cdot1/\varepsilon^d)$ charges of this type.

Assuming $d$ is constant, we have $O(1/\varepsilon^d)$ charges per $v'$, and at most $O(n/\varepsilon^d)$ pairs generated, as $\mathcal T$ has $O(n)$ nodes. Additionally, the algorithm terminates in $O(n\log n+n\varepsilon^{-d})$ time.
